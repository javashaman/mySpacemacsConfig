My spacemacs config
===================

Supports
--------
* flutter
* typescript
* react
* javascript
* html
* markdown
* yaml
* etc...

Uses 
----
* `tide` for typescript errors, etc... (`tsserver` which is basically intellisense)
* `company` for completion 
* prettify
* neotree
* osx

Settings
--------
* no auto format on save (For day job, I change this at night)
